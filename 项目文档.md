
# PasswordManager - 密码管理器

## 项目概述

PasswordManager 是一个基于 iOS 平台的密码管理应用，主要用于安全存储和管理文件与用户凭证的映射关系。项目采用 App Group 共享容器机制，实现主应用与 Share Extension 之间的数据共享。

## 核心功能

### 当前实现（第一阶段）
- 单向写入加密资产
- 文件与 LDAP 用户凭证映射存储
- SQLite WAL 模式数据库管理
- 支持记录访问时间和同步时间

### 功能规划（待扩展）
- 双向同步机制
- 文件加密传输
- 用户认证与权限管理
- 云端备份与恢复

## 技术架构

### 架构设计
采用分层架构设计，通过 App Group 实现跨进程数据共享。

```
┌─────────────────────────────────────────────────────────────┐
│                    应用层 (Application Layer)                │
│  ┌─────────────────┐              ┌───────────────────────┐ │
│  │ PasswordManager │              │ DocumentShareExtension│ │
│  │   (主应用)       │              │      (分享扩展)        │ │
│  └────────┬────────┘              └───────────┬───────────┘ │
│           │                                   │              │
│           └───────────────────┬───────────────┘              │
│                               ▼                              │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │                   数据层 (Data Layer)                    │ │
│  │          AppGroupDBManager (Shared Module)               │ │
│  │  - SQLite3 原生驱动                                      │ │
│  │  - WAL 模式事务管理                                      │ │
│  │  - 重试机制处理并发冲突                                   │ │
│  └─────────────────────────────────────────────────────────┘ │
│                               ▼                              │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │                 存储层 (Storage Layer)                   │ │
│  │  App Group Container: group.com.sean.passwordmanager     │ │
│  │  SQLite Database: file_mapping.sqlite                    │ │
│  └─────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

## 核心模块说明

### 1. AppGroupDBManager

负责管理 App Group 共享容器中的 SQLite 数据库，提供文件映射的 CRUD 操作。

**数据库表结构:**

| 字段 | 类型 | 说明 |
|------|------|------|
| file_name | TEXT | 文件名（主键，小写规范化） |
| uid | TEXT | 用户 LDAP 标识 |
| created_at | REAL | 创建时间（Julian Day） |
| last_access_time | REAL | 最后访问时间 |
| last_sync_time | REAL | 最后同步时间 |

**主要方法:**

| 方法 | 功能 |
|------|------|
| saveFileMapping | 保存文件映射记录 |
| upsertRecord | 插入或更新记录（带重试） |
| upsertRecordWithSync | 同步写入记录 |
| queryUID | 查询文件对应的用户ID |
| queryLastAccessTime | 查询最后访问时间 |
| queryLastSyncTime | 查询最后同步时间 |
| deleteRecord | 删除记录 |
| fetchAllLog | 获取所有记录日志 |

### 2. ContentView

主应用界面，提供测试入口和日志展示功能。

### 3. ShareViewController

分享扩展入口控制器，集成 SwiftUI 视图到 UIKit 环境。

## 配置与运行

### 环境要求

- macOS 14.0+
- Xcode 15.0+
- iOS 17.0+ (目标平台)

### 构建步骤

1. 打开 `PasswordManager.xcodeproj`
2. 配置开发者签名
3. 确保启用 App Groups 能力，并添加 `group.com.sean.passwordmanager`
4. 选择目标设备或模拟器
5. 运行项目

### 权限配置

两个 Target 均需配置以下 Capabilities：
- **App Groups**: `group.com.sean.passwordmanager`

## 日志规范

项目使用 Apple 官方 OSLog 框架进行日志记录：

| 日志级别 | 使用场景 |
|----------|----------|
| info | 正常操作日志 |
| error | 错误信息 |
| debug | 调试信息 |

**日志 Subsystem**: `com.sean.PasswordManager`

## 开发注意事项

1. **数据库并发**: 使用 WAL 模式和重试机制处理并发写入冲突
2. **文件名规范化**: 所有文件名存储前进行小写转换和空白字符清理
3. **错误处理**: 所有数据库操作均有完善的错误日志记录
4. **线程安全**: 建议在后台队列执行数据库操作
