Xcode项目配置说明
- 在Shared目录下创建新Swift文件后，必须在project.pbxproj中添加以下5处配置：
  1. PBXBuildFile: 添加两个 "文件名.swift in Sources" 引用（分别对应两个目标）
  2. PBXFileReference: 添加文件引用
  3. PBXGroup: 将文件添加到Shared组的children数组中
  4. PBXSourcesBuildPhase(PasswordManager): 将文件添加到主应用目标的sources
  5. PBXSourcesBuildPhase(DocumentShareExtension): 将文件添加到扩展目标的sources
- 配置完成后，执行 rm -rf build/ 清理缓存再重新编译

编译命令说明
- 清理构建缓存：rm -rf build/
- 模拟器编译分享扩展：xcodebuild -project PasswordManager.xcodeproj -target DocumentShareExtension -sdk iphonesimulator -configuration Debug build
- 模拟器编译主应用：xcodebuild -project PasswordManager.xcodeproj -target PasswordManager -sdk iphonesimulator -configuration Debug build
- 真机编译分享扩展：xcodebuild -project PasswordManager.xcodeproj -target DocumentShareExtension -sdk iphoneos -configuration Release build
- 真机编译主应用：xcodebuild -project PasswordManager.xcodeproj -target PasswordManager -sdk iphoneos -configuration Release build
